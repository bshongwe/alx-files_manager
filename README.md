<br></br>

## 📋 Objective
Build a simple platform to upload and view files:
- 🔒 User authentication via a token
- 📂 List all files
- ⬆️ Upload a new file
- 🔄 Change permission of a file
- 👀 View a file
- 🖼️ Generate thumbnails for images
<br></br>
